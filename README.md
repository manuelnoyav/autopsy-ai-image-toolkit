# Autopsy AI Image Module

Este proyecto integra capacidades de Inteligencia Artificial para el análisis automático de imágenes en investigaciones forenses mediante la herramienta Autopsy.

## Componentes

El repositorio se divide en dos partes principales:

- **`modulo/`**  
  Contiene el módulo Python para Autopsy, que extrae imágenes y se comunica con el servidor de IA.

- **`servidor/`**  
  Contiene el servidor backend desarrollado con FastAPI y Docker, encargado de procesar las imágenes mediante modelos de visión por computador.

## Requisitos

Consulta los archivos `README.md` dentro de cada carpeta (`modulo/` y `servidor/`) para conocer las instrucciones de instalación, configuración y ejecución específicas de cada componente.
